Sets whether or not the compiler should build STL modules.

```lua
buildstlmodules("value")
```

### Parameters ###

`value` is one of:

- `On`
- `Off`

### Applies To ###

The `config` scope.

### Availability ###

Premake 5.0.0 beta 3 or later for Visual Studio 2022 and later.
